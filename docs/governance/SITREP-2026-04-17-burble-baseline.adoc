== Quick Answer: "Is Burble mostly Ephapax?"

No, not by repository composition.

Based on extension counts (file-count level, excluding deps/build outputs):

* `*.ex` = 98
* `*.exs` = 38
* `*.res` = 36
* `*.zig` = 18
* `*.idr` = 12
* `*.eph` = 6

Burble has active Ephapax use, but the dominant implementation footprint is
Elixir, with AffineScript/Zig/Idris2 support layers and large AsciiDoc governance.
