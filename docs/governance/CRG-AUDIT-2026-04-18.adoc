== Verdict

*Current CRG grade: **D (Alpha)**.*

This diverges from the self-declared `C` claim. Under v2.0 strictness,
Burble is technically substantial but does not yet meet C's evidence bar.

Strong positive signals:

1. D-floor structure is present (21 workflows, canonical `.machine_readable/descriptiles/`, contractile trident).
2. Real implementation and proof surface exists (Idris2 ABI modules + Zig FFI + server/client stacks).
3. Test inventory is non-trivial and partially runnable locally (`zig build test` passed).

Immediate blockers to C:

1. Add `[dogfooding-status]` in `STATE.a2ml` with dated evidence entries.
2. Add and maintain a concrete dogfood log in `docs/practice/`.
3. Add explicit CI test jobs for server and Zig test paths, then record recent green runs.
4. Raise README orientation coverage in `server/`, `client/`, `ffi/`, and `tests/`.
5. Add `docs/READINESS.md` (or `.adoc`) aligned to v2.0 strict criteria.
