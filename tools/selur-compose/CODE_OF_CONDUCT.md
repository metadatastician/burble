# Code of Conduct

selur-compose follows the same code of conduct as the broader Burble /
hyperpolymath project. The canonical text is maintained at:

  https://github.com/hyperpolymath/burble/blob/main/CODE_OF_CONDUCT.md

## Short version

Be respectful. No harassment, no discrimination, no personal attacks.
Critique the code; don't critique the person.

Technical disagreements are resolved by the maintainers listed in `MAINTAINERS.adoc`,
whose decisions are final for this repository.

## Reporting

To report a code-of-conduct violation, email the lead maintainer at
`j.d.a.jewell@open.ac.uk` with the subject line `[selur-compose] CoC report`.
Reports are handled confidentially.
